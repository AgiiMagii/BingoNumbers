This is a simple number guessing game developed as a homework assignment. The program randomly generates 10 unique numbers between 1 and 99. Then the user is asked to input 10 unique numbers within the same range.

After all numbers are entered, the program compares user's numbers with the generated ones and calculates how many numbers are matching.

To ensure uniqueness and correctness, input validation is implemented using if conditions - to ensure that the input is a number, it's in a correct range and unique. Only valid entries are added to the lists.

I tried to structure the code to be as clean and logical as possible.
